# DIRGASTREAM

Website streaming IPTV pribadi dengan 15.000+ channel dari 178 negara.

## Fitur
- 15,322 channel live dari seluruh dunia
- Filter by negara, kategori, search real-time
- HLS player dengan stream alternatif otomatis
- Picture in Picture (PiP)
- Deteksi stream online/offline
- Offline support via Service Worker
- Favorit tersimpan di browser

## Deploy
Website ini di-host via GitHub Pages.

## Tech
- Vanilla HTML/CSS/JS
- hls.js untuk HLS streaming
- Service Worker untuk offline cache
